# SwiftUI preview app

Minimal Swift Package with a SwiftUI app to preview Turbo Themes-inspired palettes.

## Run

1. Open `examples/swift-swiftui/Package.swift` in Xcode.
2. Select the `ThemeShowcaseApp` scheme.
3. Use Canvas previews or run the app.

## Notes

- The sample uses inline palette data; swap to the official Turbo Themes Swift artifacts
  when available.
- Targets iOS 17+ and macOS 13+ for modern SwiftUI API coverage.
