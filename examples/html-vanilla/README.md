# html-vanilla

Minimal zero-dependency HTML page to preview Turbo Themes tokens and flavors.

## Run

- Open `examples/html-vanilla/index.html` directly in a browser, or
- Serve the repo root (e.g., `npx http-server .`) and visit `/examples/html-vanilla/`.

## What it shows

- Theme switching via `data-theme` on `<html>`.
- Buttons, text, and cards using Turbo token variables.
- Metrics grid to view surface/background/border tokens.
